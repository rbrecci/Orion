# Orion - Entretenimento Digital

O projeto **Orion** é uma plataforma de aluguel de filmes online, focada em oferecer uma experiência moderna, intuitiva e funcional para o consumo de entretenimento digital. Desenvolvido como parte do curso Técnico em Desenvolvimento de Sistemas na Escola SENAI "A. Jacob Lafer".

## 📅 Sprint 1: Planejamento e Design
**Período:** 12/03/2026 a 19/03/2026

Nesta primeira etapa, o foco foi a estruturação estratégica do projeto, definição de requisitos e criação da identidade visual.

### ✅ Entregas Realizadas
- **Planejamento Estratégico:** Elaboração da matriz 5W2H, cronograma detalhado e organização do fluxo de trabalho via Kanban.
- **Identidade Visual:** Definição da paleta de cores (tons de azul e roxo), tipografia (fontes *Poppins* e *Orbitron*) e criação do logotipo oficial.
- **Prototipagem:** Desenvolvimento de protótipo de alta fidelidade no Figma, incluindo telas de login, catálogo, detalhes do filme e fluxo de aluguel.
- **Documentação:** Levantamento de requisitos funcionais (login, catálogo, sistema de locação) e não funcionais (responsividade e segurança).

### 📋 Requisitos do Sistema (MVP)
* **Funcionais:**
    * Cadastro e login de usuários.
    * Visualização de catálogo de filmes.
    * Cálculo de valor de aluguel e prazos.
    * Gerenciamento administrativo de filmes e usuários.
* **Não Funcionais:**
    * Interface responsiva (Bootstrap).
    * Navegação intuitiva (UX Design).
    * Segurança na autenticação.

### 🛠️ Tecnologias e Ferramentas
- **Design & Planejamento:** Figma, Miro, Canva.
- **Front-end:** HTML5, CSS3, JavaScript e Bootstrap.
- **Back-end:** PHP.
- **Dados:** JSON para persistência de dados.
