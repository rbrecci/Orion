# Orion - Entretenimento Digital 🎬

O **Orion** é uma plataforma de locação de filmes online desenvolvida para oferecer uma experiência fluida, moderna e acessível. O projeto foca em unir um design intuitivo (UX/UI) com uma arquitetura de software sólida, permitindo que usuários naveguem por catálogos e realizem aluguéis de forma digital.

Este projeto está sendo desenvolvido no curso Técnico em Desenvolvimento de Sistemas - SENAI "A. Jacob Lafer".

## 🛠️ Tecnologias Utilizadas
* **Design:** Figma, Miro e Canva.
* **Front-end:** HTML5, CSS3, JavaScript e Bootstrap.
* **Back-end:** PHP.
* **Banco de Dados:** MySQL / JSON.

---

## 📅 Ciclo de Desenvolvimento

### 🔵 Sprint 1: Concepção e Design (Concluída)
A primeira etapa foi focada na estruturação estratégica e identidade visual do Orion.
- **Identidade Visual:** Criação de logotipo, definição de paleta de cores e tipografia (Poppins & Orbitron).
- **Documentação:** Matriz 5W2H, Cronograma e Product Backlog.
- **Prototipagem:** Desenvolvimento do protótipo de alta fidelidade no Figma, mapeando todas as telas principais (Home, Login, Cadastro, Detalhes do Filme).
- **Requisitos:** Levantamento de funcionalidades críticas (MVP).

### 🟢 Sprint 2: Desenvolvimento Back-end e Funcionalidades (Em andamento)
*Documentação em desenvolvimento...*
